# 🏦 Loan Management System (სასესხო სისტემის API)

ეს არის მარტივი სასესხო სისტემის REST API, აწყობილი **ASP.NET Core Web API**-სა და **Entity Framework Core**-ის გამოყენებით. სისტემა უზრუნველყოფს სესხების გაცემის ავტომატიზაციას, ბიზნეს ვალიდაციებს, ყოველთვიური გადასახადის (PMT) გამოთვლას, გრაფიკის შედგენასა და გადახდების აღრიცხვას.

---

## 🛠️ ტექნოლოგიური სტეკი

* **Framework:** .NET 10.0 (ASP.NET Core Web API)
* **ORM:** Entity Framework Core
* **Database:** In-Memory Database (ტესტირების სიმარტივისთვის)
* **Documentation:** Swagger UI

---

## 📈 ბიზნეს წესები & ვალიდაციები

1. **მომხმარებელი:** სესხის აღება შეუძლია მხოლოდ 18 წელზე უფროს პირს.
2. **კრედიტ სკორი:** თუ მომხმარებლის `CreditScore < 300`, სესხის განაცხადი ავტომატურად გადადის სტატუსში `Rejected` (უარყოფილია). სხვა შემთხვევაში გადადის სტატუსში `Approved`.
3. **ლიმიტები:** სესხის თანხა უნდა იყოს 500₾-დან 50,000₾-მდე, ხოლო ვადა 6-დან 60 თვემდე.
4. **PMT & გრაფიკი:** სესხის დამტკიცებისას სისტემა ფინანსური ფორმულით ავტომატურად ითვლის ყოველთვიურ გადასახადს და `LoanSchedules` ცხრილში აგენერირებს გადახდის გრაფიკს თვეების მიხედვით.
5. **გადახდები:** დახურულ ან უარყოფილ სესხზე გადახდა დაბლოკილია. გადახდის თანხა უნდა იყოს 0-ზე მეტი.
